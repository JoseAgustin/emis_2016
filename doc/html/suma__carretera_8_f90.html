program suma_carreteras

This program identifies the different Highway in the cell and adds them together.

Obtains the Highway area fraction in the cell with respect to the total municipality Highway area Highways can cover many cels, thus cels can be repeated by different higways this code reduce the number of repeated cels by addig the fraction of each highway that includes each cel.
